Resumen del Proyecto

El Problema

Los sitios web de compras en línea tienen diseños desordenados, sistemas ineficientes para ver los productos y procesos de pago confusos.

El Objetivo

Diseñar un sitio web de Tee's Shirts que sea fácil de usar, que proporcione una navegación clara y que ofrezca un proceso de pago rápido.

Investigacion sobre los usuarios: Resumen

Realicé entrevistas a los usuarios y, luego, las convertí en mapas de empatía para comprender mejor al usuario objetivo y sus necesidades. Descubrí que muchos usuarios objetivo consideran las compras en línea como una actividad divertida y relajante que realizan cuando necesitan un recreo de sus estudios o sus trabajos. Sin embargo, muchos sitios web de compras son abrumadores y confusos para navegar, y muchos usuarios objetivo se frustran por eso. Esto provocó que una experiencia normalmente agradable se convirtiera en algo desafiante para ellos y frustrara el propósito de relajación.

Investigacion sobre los usuarios: Puntos Debiles

Navegacion

Los diseños de sitios web de compras suelen estar muy cargados y eso hace que la navegación sea confusa.

Interaccion

Los botones pequeños de los sitios web de compras dificultan la selección de elementos y, a veces, eso lleva a que los usuarios cometan errores.

Experiencia

Los sitios web de compras en línea no proporcionan una experiencia de navegación atractiva.
