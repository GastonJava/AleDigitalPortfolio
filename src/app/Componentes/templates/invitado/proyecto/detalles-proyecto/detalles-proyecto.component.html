Resumen del Proyecto

El producto

Tee's Shirts es una tienda de ropa que ofrece opciones de precios asequibles. Los usuarios típicos tienen entre 19 y 30 años, y la mayoría son estudiantes universitarios o profesionales recién recibidos. rápido y fácil para cualquier tipo de usuario.

Duracion del Proyecto

Desde mayo 2022 hasta agosto 2023

Resumen del Proyecto

El Problema

Los sitios web de compras en línea tienen diseños desordenados, sistemas ineficientes para ver los productos y procesos de pago confusos.

El Objetivo

Diseñar un sitio web de Tee's Shirts que sea fácil de usar, que proporcione una navegación clara y que ofrezca un proceso de pago rápido.

Resumen del Proyecto

Mi Funcion

Diseñador de UX que lidera el diseño del sitio web de Tee's Shirts.

Responsabilidades
